How to run Online Book Store using PHP and MySQL

1. Download the zip file

2. Extract the file and copy obsms folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/HTML)

4.Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with the name obsmsdb

6. Import obsmsdb.sql file(given inside the zip package in the SQL file folder)

7. Run the script http://localhost/obsms
